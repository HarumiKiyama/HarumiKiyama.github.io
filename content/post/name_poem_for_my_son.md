岁次昭阳单阏，十一月乙巳，子鸣谦生。 <br/>
鸣谦，在周易有之，谦之升，曰鸣谦贞吉。处正得中，行谦广远，以中和为心，正而得吉也。 <br/>
念我平生，少顽愚多行崎路，而今震来哑哑，不修阴骘而有此报，真不可解也。 <br/>
因思, <br/>
素王受命，谓我心忧。 <br/>
孝武燕乐，将奈老何。 <br/>
赤舄金縢，其情非虚。 <br/>
儒仲误子，安为高士。 <br/>
百感俱下，聊作古风以志之，其辞为。 <br/>

猗猗秋兰，植于庭宇。 <br/>
绵绵瓜瓞，蔓于斗女。 <br/>
烂烂岩电，轩轩霞举。 <br/>
卓卓野鹤，肃肃独立。 <br/>
奋翮振秽，持心勿逸。 <br/>
被褐怀玉，鼓琴蓬荜。 <br/>
明夷于飞，登天入地。 <br/>
有兔爰爰，将寘丛棘。 <br/>
二鼠为虐，三彭作厉。 <br/>
蒿里催逼，安可久庇。 <br/>
吐哺含饴，心忧可易。 <br/>
易篑之际，幸不我弃。
